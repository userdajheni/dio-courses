O que é HTML?

O HTML (HyperText Markup Language) é a linguagem padrão usada para estruturar páginas na web. Em português, significa Linguagem de Marcação de Hipertexto, e sua principal função é organizar o conteúdo de um site em elementos como títulos, parágrafos, links, imagens, tabelas e muito mais.

Ele não é uma linguagem de programação, mas sim de marcação, porque utiliza “tags” (ou etiquetas) para indicar como cada parte do conteúdo deve ser exibida no navegador.

O HTML foi criado no início dos anos 1990 por Tim Berners-Lee, cientista britânico considerado o pai da World Wide Web. A primeira versão foi lançada em 1991, e, desde então, o HTML passou por várias evoluções, chegando atualmente ao HTML5, que é a versão mais moderna e completa, trazendo novos recursos como suporte para áudio, vídeo e elementos semânticos que facilitam a acessibilidade e a organização da informação.

Hoje, o HTML é uma das três tecnologias fundamentais da web, junto com o CSS (responsável pelo estilo) e o JavaScript (responsável pela interatividade). Sem ele, simplesmente não existiria internet como conhecemos.

O que são elementos HTML?

Os elementos HTML são os blocos de construção de qualquer página web. Cada elemento geralmente é formado por uma tag de abertura, um conteúdo e uma tag de fechamento. Exemplo:

< p > Este é um parágrafo < /p >

  • < p > é a tag de abertura;
  • Este é um parágrafo é o conteúdo;
  • < /p > é a tag de fechamento.

Alguns elementos, como < img > ou < br >, não precisam de tag de fechamento. Os elementos servem para estruturar a página: título, texto, imagem, link, lista, botão… tudo é um elemento!
