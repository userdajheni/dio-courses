Links no HTML

Se a web fosse uma cidade, os links seriam as ruas que conectam cada lugar. É clicando neles que a gente “viaja” de uma página pra outra.

O que é um link?

No HTML, links são criados com a tag <a> (de "anchor", âncora). Sintaxe básica:

<a href="URL">Texto do Link</a>

Atributo target

O atributo target define onde o link vai abrir:

  • _self → abre na mesma aba (padrão)
  • _blank → abre em nova aba
  • _parent → abre no frame pai
  • _top → ocupa a janela inteira

Exemplo: Abrir o Google em nova aba

URLs Absolutas vs Relativas

- Absoluta: link completo (https://...).
- Relativa: link para uma página dentro do mesmo site.
