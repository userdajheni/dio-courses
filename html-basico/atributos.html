Atributos em HTML

Atributos fornecem informações extras para os elementos HTML. Eles sempre aparecem na tag de abertura e geralmente seguem o formato: nome="valor".

Boas práticas

  • Sempre use letras minúsculas nos atributos.
  • Sempre coloque aspas em torno dos valores.
  • Prefira aspas duplas, mas aspas simples também funcionam.

Exemplos de atributos importantes

href src width e height alt
Define o destino de um link Usado para indicar o caminho de uma imagem Definem o tamanho da imagem em pixels Fornece um texto alternativo se a imagem não puder ser carregada (muito útil para acessibilidade).

style lang title
Permite adicionar estilos diretamente no elemento Define o idioma da página dentro da tag <html>. Mostra uma dica (tooltip) ao passar o mouse

Resumo

  • Todos os elementos HTML podem ter atributos.
  • href: destino de links.
  • src: caminho da imagem.
  • width e height: tamanho da imagem.
  • alt: texto alternativo de imagem.
  • style: estilos inline.
  • lang: idioma da página.
  • title: informação extra em tooltip.
